# 繁中生圖工坊 v1.1 更新說明

## 修復內容

### 🔧 1. 圖片載入失敗（全面重寫）

**問題**：Ideogram、FLUX、SD 3.5、Z-Turbo、Recraft 模型都顯示「圖片載入失敗」

**修復**：使用 4 種方式依序嘗試載入：

| 順序 | 方式 | 說明 |
|------|------|------|
| 1 | 直接載入 | 使用 crossOrigin，部分 CDN 支援 |
| 2 | wsrv.nl | 專業圖片代理服務 |
| 3 | images.weserv.nl | 專業圖片代理服務 |
| 4 | corsproxy.io | 通用 CORS 代理 |

- 每種方式 25 秒超時
- 使用專業圖片代理服務（比通用代理更穩定）

---

### ✏️ 2. AI 解析結果可編輯

- `<div>` → `<textarea>`
- 可在按「使用」之前修改內容
- 新增 focus 狀態樣式

---

### 📱 3. Qwen-Image 改用雙引號

**改進**：手機上更容易輸入

**舊寫法** ❌
```
一隻柴犬，招牌寫著「新年快樂」
```

**新寫法** ✅
```
一隻柴犬，招牌寫著"新年快樂"
```

**其他優化**：
- 自動加入中文優化後綴：`超清，4K，电影级构图`
- 強制請求 `b64_json` 格式（避免 CORS 問題）
- 提高生成步數：20 → 28
