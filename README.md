# 繁中生圖工坊 v1.3 更新說明

## 🔧 1. 圖片載入邏輯重寫

**新策略**：使用 Image 物件測試載入，無需 CORS 代理轉換

```
1. 直接載入原始 URL（8秒超時）
   ↓ 失敗
2. wsrv.nl 圖片代理
   ↓ 失敗
3. images.weserv.nl 圖片代理
   ↓ 失敗
4. imageproxy.pimg.tw 圖片代理
```

**優點**：
- Image 物件載入不受 CORS 限制
- 只要瀏覽器能顯示，就算成功
- 下載時才需要代理轉換

---

## 💡 2. 新增 AI 提示詞建議功能

**使用方式**：
1. 在描述欄輸入簡短描述（如：柴犬 新年）
2. 點擊「💡 AI建議」按鈕
3. AI 會生成 5 個優化後的完整提示詞
4. 點擊任一建議即可套用

**範例**：
- 輸入：`柴犬 招牌 新年`
- AI 建議：
  - 可愛柴犬坐在日式商店前，木製招牌上寫著"新年快樂"四個金色大字...
  - 毛茸茸的柴犬戴著紅色圍巾，背景是新年裝飾...
  - ...

## 🧠 3. Qwen-Image 中文優化

**正確用法**：
```
可愛柴犬坐在店門口，紅色招牌上寫著"新年快樂"四個金色大字
```

用 **雙引號 `""`** 包住要顯示的文字！

**自動優化**：
- 偵測到引號文字 → 加入 `高清晰文字渲染, 文字清晰可讀, 正確的中文字形`
- 偵測到中文 → 加入 `超清, 4K, 电影级构图`
- 強制 `response_format: b64_json`
- 步數：30
