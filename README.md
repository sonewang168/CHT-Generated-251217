## 🀄 文字疊加使用方法

1. 上傳圖片
2. 點擊 **AI → 🀄文字疊加**
3. 輸入繁體中文文字
4. 選擇 **橫書** 或 **直書**
5. 選擇位置：
   - **九宮格**：點擊按鈕快速定位
   - **自由拖曳**：勾選後拖曳到任意位置
6. 點擊「套用」
